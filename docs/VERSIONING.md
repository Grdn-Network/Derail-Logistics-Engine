# Versioning

Versions are `MAJOR.MM.PPP`, tags are `vMAJOR.MM.PPP`.

- **MAJOR**: `0` is pre-release. `1` is release. Anything higher is an overhaul
  of something fundamental.
- **MM** (two digits): progress toward the next major. The tens digit moves on
  milestone waves (`0.40` is the economy wave, `0.50` the community beta); the
  ones digit moves on meaningful but smaller additions inside a wave.
- **PPP** (three digits): bug fixes, tests and other small work.

History before the scheme maps directly: `0.1.x` became `0.1x.000`, `0.2.0`
became `0.20.000`, `0.3.0` became `0.30.000`.
